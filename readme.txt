**Cooper Consulting Scheduling Application**
V. 1.0
Jan 20, 2021
Author: Sean Cooper
Contact: scoo182@wgu.edu

The purpose of the application for consultants to be able to schedule appointments with clients at all 4 Country locations.

IDE Used: 
IntelliJ IDEA 2020.3.1 (Community Edition)
Build #IC-203.6682.168, built on December 29, 2020
Runtime version: 11.0.9.1+11-b1145.63 amd64
VM: OpenJDK 64-Bit Server VM by JetBrains s.r.o.

Directions:
1. Open Program and Log in
2. Appointment Screen: Appoitments can be viewed here as well as buttons to add and modify appoitments. 
3. Manage Customers: Customers are viewed here. Customers can be added or modified here.
4. Adding/Modifying Appointments/Customers: All information must be filled out. An error occurs when information is missing.
5. Reports. Their are three reports, two appointment reports and one customer report
	- Monthly Appointment Report allows the user to view appointments by month and type and displays the total number.
 	- Conact Appointment Report allows the user to view appointments by conact and displays the total number.
	- Country Cusotmer Report allows the user to view customers by country and displays the total number.
